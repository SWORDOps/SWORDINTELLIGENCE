## Data Security

We implement enterprise-grade security measures:

### Technical Controls
- **Post-Quantum Encryption:** ML-KEM-1024 + ML-DSA-87 (NIST FIPS 203/204)
- **AES-256-GCM:** Authenticated encryption for data at rest and in transit
- **Hardware Security:** Support for YubiKey, CAC cards, and HSM integration
- **Multi-Factor Authentication:** TOTP and WebAuthn/FIDO2
- **Secure WebSockets:** Encrypted real-time communications

### Operational Controls
- Background checks for all personnel with data access
- Strict access controls and audit logging
- Incident response procedures
- Regular security assessments

### Data Retention
- Contact inquiries: 7 years (or until deletion request)
- Client engagement data: Per contractual obligations (typically 7 years minimum)
- Audit logs: 7 years for compliance
